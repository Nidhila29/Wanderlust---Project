📌 Project Title

WanderLust – Full Stack Travel Listing Web App

📖 Description

A full-stack MERN application inspired by Airbnb that allows users to explore, create, and manage travel listings with authentication, image uploads, and search functionality.

🚀 Live Demo

🔗 Deployed Link: https://wanderlust-project-v417.onrender.com

🛠️ Tech Stack

Frontend: [HTML, CSS, Bootstrap, EJS],
Backend: [Node.js, Express.js],
Database: [MongoDB Atlas],
Authentication: [Passport.js],
Other Tools: [Cloudinary, Leaflet]

✨ Features

User authentication (login/signup),
Create, edit, and delete listings,
Image upload with Cloudinary,
Search listings by title/location,
Responsive UI,
Secure routes & validation

👉Future Improvements

Implement advanced search and sorting filters,
Add pagination for large numbers of listings,
Improve UI/UX and mobile responsiveness,
Optimize database queries for better performance
